# ttrpgCharacterStorage
Java based program to create and store characters for Daybreak ttrpg system

## Requirements
- Should allow users to create characters in a step by step process, allowing for level ups later on 
- Should perform calculations on character stats to display derived stats like spell attack or saves
- Can save characters and manage different characters files, files should use zipped folders to facilitate saving of images alongside characters
- Architecture should allow for easy changes to stat algorithms in the event of rules changes
- Dictionary of spells and items which can be searched or used for tooltips
- Include installer solution so that users do not have to build program themselves
- Tooltips that inform users of the calculations performed
- Potential dice rolling system to assist with online play
